## 🔧 Como Aplicar as Imagens

### Opção 1: Download Manual

1. Acesse cada URL acima
2. Clique com botão direito → "Salvar imagem como"
3. Salve em `public/images/estilos/` com o nome correto:
   - `minimalismo.webp`
   - `classico.webp`
   - etc.

### Opção 2: Script Automático (Node.js)

Criado arquivo: `download-style-images.js`

**Como usar:**
```bash
npm install node-fetch
node download-style-images.js
```

### Opção 3: Script Automático (Python)

Criado arquivo: `download-style-images.py`

**Como usar:**
```bash
pip install requests pillow
python download-style-images.py
```

## 🎨 Alternativas para Imagens Melhores

### Projetos Reais WG Almeida:
- Fotografar projetos reais da empresa
- Mais autenticidade
- Portfólio próprio

### Banco de Imagens Premium:
- **Shutterstock** (pago)
- **Getty Images** (pago)
- **Adobe Stock** (pago)
- Qualidade superior e exclusividade

### IA (Geração):
- **Midjourney** - Gerar imagens específicas por estilo
- **DALL-E** - Criar renders realistas
- **Stable Diffusion** - Customização total
