# Galeria de Imagens

Coloque aqui todas as imagens para galerias de projetos e portfólio.

## Organização sugerida:

Crie subpastas por projeto ou categoria:

```
gallery/
├── arquitetura/
│   ├── projeto-1.jpg
│   ├── projeto-2.jpg
│   └── ...
├── engenharia/
│   ├── obra-1.jpg
│   ├── obra-2.jpg
│   └── ...
└── marcenaria/
    ├── mobiliario-1.jpg
    ├── mobiliario-2.jpg
    └── ...
```

## Especificações:
- Formato: JPG (para fotos) ou PNG (para imagens com transparência)
- Resolução: 1200x800 ou superior
- Tamanho: 300-800 KB por imagem (otimizado)
- Qualidade: 80-90%

## Dicas:
- Nomeie os arquivos de forma descritiva
- Organize por categorias
- Mantenha proporções consistentes
- Otimize todas as imagens para web
