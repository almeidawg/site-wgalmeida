⚙️ Tecnologia

Stack Técnico Moderno

⚛️

Frontend

React 19 + Vite + Tailwind CSS + Zustand (state management)

🗄️

Backend

Express.js (Node.js) + TypeScript + PostgreSQL via Supabase

🚀

Deploy

Vercel (Frontend) + Railway (Backend). CI/CD com GitHub Actions.

🔗

Integrações

Google Calendar, Slack, Zapier, Google Drive, APIs customizadas.

📊

Analytics

Google Analytics 4, dashboards BI, eventos customizados.

🔐

Segurança

Auth (SSO/OAuth), RLS (Row Level Security), criptografia end-to-end.
