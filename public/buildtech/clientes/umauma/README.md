# 🎯 Projeto UMAUMA — EventOS

**Sistema Operacional para Ecossistemas de Eventos**

> WG build.tech × grupo UMAUMA

---

## Sobre o Projeto

Plataforma de gestão integrada para o grupo UMAUMA — ecossistema de 14 agências de entretenimento, eventos, live marketing, música, conteúdo, branding e hospitalidade sediado em São Paulo.

- State: Zustand
- Monorepo: Turborepo
