# Vídeos de Abertura do Site

Coloque aqui os vídeos de abertura do site.

## Arquivos necessários:

- **hero-desktop.mp4** - Vídeo horizontal para computadores e tablets (landscape)
- **hero-mobile.mp4** - Vídeo vertical para celulares (portrait)

## Especificações recomendadas:

### Vídeo Desktop (hero-desktop.mp4)
- Orientação: Horizontal (16:9)
- Resolução recomendada: 1920x1080 ou 1280x720
- Formato: MP4 (H.264)
- Tamanho máximo: 10-20 MB
- Duração: 10-30 segundos (com loop)

### Vídeo Mobile (hero-mobile.mp4)
- Orientação: Vertical (9:16)
- Resolução recomendada: 1080x1920 ou 720x1280
- Formato: MP4 (H.264)
- Tamanho máximo: 5-10 MB
- Duração: 10-30 segundos (com loop)

## Dicas:
- Use vídeos otimizados para web (comprimidos)
- Evite vídeos muito longos
- Teste em diferentes dispositivos
